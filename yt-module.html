Shadowrocket YT модуль

Настройка модуля YT-Premium-V1-RU

Подключение модуля YouTube в Shadowrocket для просмотра YouTube без рекламы

← Назад
1

Скопируй ссылку модуля

https://raw.githubusercontent.com/misha-tgshv/shadowrocket-configuration-file/refs/heads/release/modules/YT-Premium-V1-RU.module

2

Открой Shadowrocket

Перейди в Config → Modules


3

Добавление модуля

Нажми + → вставь ссылку → Download


4

Включение модуля

Нажми на скачанный модуль и на кнопку Enable Module


5

Включение HTTPS Decryption, выпуск и установка сертификата

1️⃣ Перейди на вкладку Config и напротив нужной конфигурации нажмите ℹ️;

2️⃣ Перейди в раздел HTTPS Decryption;
3️⃣ Включи переключатель HTTPS Decryption;
4️⃣ Нажми на кнопку Certificate;
5️⃣ Нажми Generate A New CA Certificate;
6️⃣ Подтверди действие, снова выбрав Generate A New CA Certificate;
7️⃣ После генерации выбери Install CA Certificate to System;
8️⃣ В появившемся окне выбери Allow (Разрешить);
9️⃣ Появится сообщение Profile Downloaded (Профиль загружен).

6

Установка профиля сертификата

1️⃣ Открой Настройки на iPhone;
2️⃣ В самом верху появится пункт Profile Downloaded (Загружен профиль) — нажми на него;

3️⃣ Нажми Install (Установить), введи пароль от устройства;
4️⃣ Снова нажми Install (ещё раз подтверди);
5️⃣ Сертификат будет установлен (Profile Installed).

7

Сделай сертификат доверенным

1️⃣ Открой Настройки → Основные → Об этом устройстве → в самом низу выбери Certificate Trust Settings (Доверие сертификатам);
2️⃣ Включи переключатель напротив твоего сертификата Shadowrocket;

3️⃣ Подтверди действие, нажав Continue (Продолжить).

8

Перезапуск YouTube

Полностью закрой приложение YouTube и открой снова для применения правил.

9

Автообновление модуля

Перейди в приложении Shadowrocket в Settings → Modules → Auto Background Update → Вкл.
